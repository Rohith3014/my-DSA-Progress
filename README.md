## 📚 Topics Covered

### 🔹 If-Else Statements

Conditional statements are used to make decisions in programs based on certain conditions.

**Concepts Learned:**

* Understanding conditional logic
* Even / Odd number checking
* Range-based conditions
* Decision-making in programs

**Example:**

```python
num = 10

if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

## 🔁 Loops in DSA (For & While)

### 📖 Introduction

Loops are one of the most important concepts in Data Structures and Algorithms (DSA).
They help in repeating a block of code efficiently and are used in almost every problem.

---

### 🔹 Types of Loops

### 1️⃣ while Loop

Used when the number of iterations is unknown and depends on a condition.

**Use Cases:**

* Condition-based looping
* When iteration count is unknown
* Infinite loops with break

**Key Points:**

* Executes as long as the condition remains true
* Useful for input validation and dynamic conditions
* Requires manual update of loop variable to avoid infinite loops
* Commonly used in searching, simulations, and iterative problems

**Example:**

```python
i = 1

while i <= 5:
    print(i)
    i += 1
```

---

### 2️⃣ for Loop

Used when the number of iterations is known.

**Use Cases:**

* Iterating over a range
* Traversing lists, strings, arrays

**Key Points:**

* Cleaner and more readable than while loops
* Automatically handles iteration
* Widely used in DSA problems

**Example:**

```python
for i in range(1, 6):
    print(i)
```
